用github.io搭建hexo博客

  • github
  • hexo

全局安装hexo
npm install -g hexo #全局安装hexo
hexo init myblog #新建myblog文件夹，并初始化
npm install #安装所需的所有hexo模块
$ hexo new “My New Post” #新建博文 hexo n
$ hexo server #启动本地4000端口服务器 hexo s
$ hexo generate #编译生成静态HTML文件 hexo g
$ hexo deploy #部署到远程网站 hexo d

在Github
1 创建repository hanwisdom.github.io
SSH地址应为 git@github.com:hanwisdom/hanwisdom.github.io.git

2 ssh-keygen -t rsa -C “$your_email”,一直按enter。 #SSH公钥私匙获取
cd ~/.ssh #进入公私钥文件夹
cat id_rsa.pub #直接显示公钥的全部字串码
新建SSH公钥，把全部字串码复制到SSH公钥

3 $ ssh -T git@github.com #输入命令应显示如下
Warning: Permanently added the RSA host key for IP address ‘192.48.252.150’ to

the list of known hosts.
Enter passphrase for key ‘/c/Documents and Settings/Administrator/.ssh/id_rsa’:
再输入密码，应显示如下
Hi hanwisdom! You’ve successfully authenticated, but GitHub does not provide

shell access.

4发布到github
打开Hexo本地文件夹的“_config.yml”，编辑以下代码
deploy:
type: github
repository: git@github.com:hanwisdom/hanwisdom.github.io.git
branch: master

5 echo “# hanwisdom.github.io” >> README.md
git init
git add README.md
git commit -m “first commit”
git remote add origin git@github.com:hanwisdom/hanwisdom.github.io.git
git push -u origin master

6 npm install hexo-deployer-git –save
hexo generate –deploy #每次修改本地文件后，必须generate编译保存再远程部署。

7 打开https://github.com/hanwisdom/hanwisdom.github.io #显示代码上传成了
游览器打开http://hanwisdom.github.io/ #出现你的博客

8 如何更换主题theme
github.com 搜索hexo-theme,选择light主题为例，下载zip,解压到本地themes下，简化文

件夹名字，再在_config.yml中相应位置改为theme: light。

也可以命令行 git clone https://github.com/zippera/light.git themes/light

9 全自动写博客
1.在本地安装pm2(一个可在后台永久打开一个node小程序的 nodejs应用，也可以监控文件

变化)
sudo npm install -g pm2
2.在 博客的source下新建 start.js，内容如下
var process = require(‘child_process’);

process.exec(‘ hexo g -d’, function (error, stdout, stderr) {
if (error !== null) {
console.log(‘exec error: ‘ + error);
}
});
3.在同级目录下创建 watch.json,内容如下
{
“apps” : [{
“name” : “blog”,
“script” : “./start.js”,
“exec_interpreter”: “node”,
“exec_mode” : “fork_mode”,
“watch” : “_posts”
}]
}
4.使用pm2命令实现监控文件变动自动提交
pm2 start watch.json
你可以把_post目录创建一个快捷方式到桌面，以后写博客直接打开文件夹进行新建文件
修改文件，即可实现自动上传
